# 若依管理系统 - 快速启动指南

## 环境要求

- Java 17+
- MySQL 8.0
- Node.js 18+
- Maven 3.8+（可选，已提供 Maven Wrapper）

## 一、启动后端

### 方式1：使用一键启动脚本（推荐）

```powershell
powershell -ExecutionPolicy Bypass -File start.ps1
```

### 方式2：Maven Wrapper 启动

```powershell
$env:JAVA_HOME = 'D:\JDK'
.\mvnw.cmd spring-boot:run -f ruoyi-admin
```

### 方式3：Maven 启动（需安装Maven）

```powershell
mvn spring-boot:run -f ruoyi-admin
```

### 后端服务地址

- API地址：http://localhost:8080/
- 启动成功标志：控制台显示 `若依启动成功`

---

## 二、启动前端

### 一键启动脚本已包含前端

### 手动启动

```powershell
cd ruoyi-ui
npm run dev
```

### 前端服务地址

- 访问地址：http://localhost:80/
- 启动成功标志：控制台显示 `Compiled successfully`

---

## 三、停止服务

```powershell
powershell -ExecutionPolicy Bypass -File stop.ps1
```

---

## 四、验证服务状态

```powershell
# 检查后端端口
netstat -ano | Select-String ":8080"

# 检查前端端口
netstat -ano | Select-String ":80"

# 测试后端API
Invoke-WebRequest -Uri "http://localhost:8080/" -UseBasicParsing
```

---

## 五、常见问题

| 问题 | 解决方案 |
|---|---|
| 端口被占用 | 运行 stop.ps1 后重新启动 |
| 数据库连接失败 | 检查MySQL服务是否启动，账号密码是否正确 |
| 前端代理错误 | 检查 `ruoyi-ui/vue.config.js` 中的 `proxy.target` |
| npm install 失败 | 使用 `npm install --registry=https://registry.npmmirror.com` |
| 首次启动慢 | Maven Wrapper 需下载依赖，等待几分钟 |
